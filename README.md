## ğŸ“¦ Getting Started

### Prerequisites

Make sure you have:

- Node.js v18+
- npm or yarn

### Installation

```bash
git clone https://github.com/vinayaksandilya/Meta-Trader-UI.git
cd Meta-Trader-UI
npm install
# or
yarn install
````

### Development Server

```bash
npm run dev
# or
yarn dev
```

App will be running at [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
```
